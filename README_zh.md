# 聊天记录生成器

可高度定制的聊天记录生成工具。面向科研场景，用于准备实验材料或问卷中使用的对话截图。

## 快速开始

1. 用浏览器打开 `聊天记录生成器.html`
2. 左侧面板添加消息、调节样式，右侧实时预览
3. 点击「导出 JSON」保存配置和消息，下次打开可导入恢复
4. 也可以从 `templates/` 目录导入已有场景直接开始

## 功能

- **消息类型**：文字消息、图片消息、系统消息
- **样式独立调节**：对方/己方各自控制头像显隐/大小/位置、昵称全套、时间戳、气泡颜色/圆角/阴影/箭头、文字字号/颜色/行高/对齐/加粗/斜体
- **布局预设**：左右布局、全部靠左、全部靠右、居中对称、纯文本流、交错排列（一键切换，不改样式）
- **顶部栏可配**：名称文字/字号/颜色/位置、头像显隐/位置/上传
- **聊天区可配**：背景色、预览宽度（300–900px 或自适应）
- **JSON 导出/导入**：完整保存配置 + 消息，方便实验复现和版本管理
- **选中操作**：点击任意消息选中后可编辑或删除

## 科研用途

对话截图作为实验材料在以下领域常见：

- 消费者行为（客服对话、商品咨询）
- 人机交互（AI 助手、聊天机器人评估）
- 教育技术（在线辅导、同伴对话）
- 社会心理学（人际沟通、情绪表达）

工具将所有视觉参数暴露为独立控件，研究者可以精确控制每一处细节，确保实验条件之间的差异仅来自预设变量。导出 JSON 文件可直接附在论文补充材料中，实现完全可复现。

## 技术说明

纯静态 HTML（CSS + JavaScript），无框架、无依赖（图标库可选）。数据驱动渲染，配置变更即时刷新预览。
